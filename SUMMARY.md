## ğŸ’¡ Usage Tips

### Quick Start
```typescript
import { createMimic } from 'mimic-data';

// Use any alias!
const mimic = createMimic('jp'); // Same as 'ja_JP' or 'japan'
console.log(mimic.identity.fullName()); // "Tanaka Kenji"
```

### Multi-Locale Application
```typescript
const locales = ['us', 'jp', 'de', 'br', 'in'];
const users = locales.map(locale => {
  const mimic = createMimic(locale);
  return {
    locale,
    ...mimic.identity.person(),
    ...mimic.location.address()
  };
});
```

### Finding Available Codes
```typescript
import { getAllLocaleCodes } from 'mimic-data';

console.log(getAllLocaleCodes());
// ['en_US', 'en', 'us', 'usa', 'ja_JP', 'ja', 'jp', 'japan', ...]
```
